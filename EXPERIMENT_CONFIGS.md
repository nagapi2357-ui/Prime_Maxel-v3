# V3 Experiment Configurations

**Purpose:** Test whether the 1/2 subharmonic and mirror symmetry affect resonance quality.
**Method:** Same board, different GreenPAK programming. Compare results.

---

## Config A — Original (1/1 as centre node)

```
N1    N2    N3    N4    N5    N6   │  N7    N8    N9    N10   N11   N12
1/13  1/11  1/7   1/5   1/3   1/1  │  2/1   3/1   5/1   7/1   11/1  13/1
```
- 1/1 is a physical node (unity frequency source)
- 1/2 absent — no octave subharmonic
- Asymmetric: macro has 2, micro doesn't

## Config B — True Mirror (1/1 as virtual axis)

```
N1    N2    N3    N4    N5    N6   ║  N7    N8    N9    N10   N11   N12
1/13  1/11  1/7   1/5   1/3   1/2  ║  2/1   3/1   5/1   7/1   11/1  13/1
                                    ║
                               1/1 = reference (implicit)
```
- Every prime has both harmonic and subharmonic
- 1/1 is the mirror axis — measured against, not generated
- Perfect symmetry: N6↔N7 are octave partners (1/2 ↔ 2/1)

### Execution order:

1. **Config A first** — this matches the original design intent, gives us a baseline
2. **Config B second** — reprogram GreenPAKs with 1/2 replacing 1/1
3. **Config A again** — re-run to confirm repeatability (rules out thermal drift / setup changes)

A-B-A pattern eliminates ordering bias.

### What to watch for:

- **Q_ℓ difference** — does true symmetry sharpen or blur the constant?
- **Phase coherence** — does the octave pair (1/2 ↔ 2/1) create cleaner phase relationships?
- **Revival pattern** — does the pulse propagate differently without a unity source?
- **Anything unexpected** — this is where discovery lives

### Base frequency note:

In Config B, the "1/1" reference frequency still exists — it's just implicit. All ratios are still defined relative to f₀. The difference is whether f₀ is *physically generated by a node* or *mathematically implied by the ratio structure*.
